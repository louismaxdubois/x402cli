# Security Policy

## Supported versions

| Version | Supported |
|---------|-----------|
| 0.2.x (current) | ✅ Website + demo API |
| < 0.2 | ❌ |

## Reporting a vulnerability

If you discover a security issue related to **x402cli.xyz** or **api.x402cli.xyz**:

1. **Do not** open a public GitHub Issue for sensitive reports
2. Contact [@x402cli on X](https://x.com/x402cli) via DM if available, or open a private security advisory on GitHub if enabled

## Scope notes

- Demo API uses placeholder `payTo` addresses — not a live payment system
- Website submit form does not transmit data to a backend
- No user accounts or stored credentials on the current site

## Out of scope

- Social engineering
- Issues requiring physical access to hosting panel
- Scanner mock behavior (known frontend simulation)

We appreciate responsible disclosure.
