# Contributing to x402 CLI

Thank you for your interest. This is an early-stage experimental project.

## Ways to contribute

1. **Open an Issue** — bugs, ideas, registry interest, documentation fixes
2. **Improve docs** — PRs to markdown files in this repo
3. **Share feedback** — [@x402cli on X](https://x.com/x402cli)

## What we accept

- Documentation improvements (README, docs/)
- Issue reports with clear reproduction steps
- Suggestions for CLI behavior and agent output format
- API registry metadata proposals

## What we don't accept

- Token / presale / investment related content
- Claims of official Coinbase or Base affiliation
- Spam or low-effort promotional PRs

## Website source

The deployable website (HTML/CSS/JS) is **not** in this repository. Doc changes here should mirror or link to [x402cli.xyz](https://x402cli.xyz) where relevant.

## Pull request process

1. Fork the repository
2. Create a branch (`docs/fix-typo`, `feature/issue-12`)
3. Keep PRs focused and small
4. Write clear commit messages in English
5. Open PR against `main`

## Code of conduct

Be respectful and developer-focused. No harassment or bad-faith engagement.

## Questions?

Open an Issue or reach out on X: [@x402cli](https://x.com/x402cli)
