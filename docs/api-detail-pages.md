# API Detail Pages

API detail pages describe each x402-powered API using structured metadata.

Each detail page may include:

- API name
- category
- endpoint
- network
- asset
- price
- payment protocol
- MCP-ready status
- Builder Code status
- attribution standard
- status
- description
- example curl request
- example HTTP 402 response
- registry metadata JSON
