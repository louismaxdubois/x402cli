# MPP / Tempo Research

x402 CLI currently focuses on x402-style HTTP 402 payment flows on Base.

MPP / Tempo compatibility is being tracked as future research to understand how machine payment systems could map into API discovery, paid API metadata, and agent-ready workflows.

This is not a live feature.

## MPP context

**MPP** (Machine Payments Protocol) and **Tempo** are ecosystem concepts for machine-to-machine payments. x402 CLI does not implement or endorse any specific MPP stack today.

## Safe positioning

> x402 CLI currently focuses on x402-style API discovery and HTTP 402 inspection on Base. MPP / Tempo compatibility is listed as future research, not a live feature.
